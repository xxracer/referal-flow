# **App Name**: ReferralFlow Central

## Core Features:

- Referral Form Submission: Allow referrers to submit patient information and documents through a web form.
- Automatic Referral ID Generation: Generate a unique Referral ID upon submission for tracking purposes. Format: [PREFIJO]-[AÃ‘O]-[SEQ]
- Automated Notifications: Send immediate confirmation emails to both the internal team and the referring party upon referral submission, containing the Referral ID and status.
- Fax Processing Integration: Integrate a fax service to convert faxes to PDFs and extract relevant data for processing. Implement manual handling for faxes without Referral IDs.
- Internal Management Portal: Provide a dashboard for staff to view, update, and manage referrals, including changing statuses, adding internal notes, and accessing attached documents.
- Public Status Check: Enable referrers to check the status of their referral using the Referral ID and patient's date of birth.
- Smart Categorization: AI-powered tool that analyzes the referral documents and suggests relevant categories for faster processing and assignment.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE), providing a sense of trust and efficiency.
- Background color: Light, desaturated blue (#EBF4FA), offering a clean and professional backdrop.
- Accent color: A gentle violet (#B284BE), used sparingly for calls to action.
- Headline font: 'Belleza' sans-serif font, to create personality.
- Body font: 'Alegreya' serif font, for a friendly and elegant user experience. It pairs very well with 'Belleza'.
- Use clean and clear icons to represent different referral statuses and actions.
- Maintain a clean and organized layout to facilitate easy navigation and information retrieval.
